About The Project

Understanding our fake news detection system

How It Works


Our system uses a Convolutional Neural Network (CNN) specifically designed for text classification. The model analyzes patterns in news articles by converting text into numerical representations and identifying features that distinguish fake news from real news.



Model Architecture


Embedding Layer Converts words to 128-dim vectors
Conv1D Layer 128 filters, kernel size 5
Global Max Pooling Reduces dimensionality
Dense Layer 64 neurons with ReLU
Dropout 50% dropout for regularization
Output Layer Sigmoid activation

Training Data


The model is trained on a comprehensive dataset containing:

    20,000+ fake news articles

    20,000+ real news articles

    Diverse topics and sources

    Balanced class distribution
